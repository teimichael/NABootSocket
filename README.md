# NABootSocket
Websocket system for NA Boot Plan
>Test version

> Test client path: 
>> ./NABootSocketClient


## Component
- Spring Boot
- Spring MVC
- Spring Data JPA
- Thymeleaf
- Spring WebSocket
- Spring Boot Dev Tools
- MySQL connector
- Lombok
- JUnit
- Swagger

## Command

### Start up container by docker-compose
```bash
docker-compose up
```

### Recreate container
```bash
docker-compose up --force-recreate
```
